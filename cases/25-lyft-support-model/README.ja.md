**導入。** Lyftは応答の正確さとブランドらしい語り方を比較し、より配慮が必要な業務は人が担う形でサポートに導入しました。

**FDEの示唆。** 語調、共感、エスカレーション規則も評価セットに入れます。Anthropicは解決時間87% 超の短縮、判断精度30% 超の向上を公表しています。
