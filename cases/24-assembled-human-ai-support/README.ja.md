# 24. Assembled：サポートエージェントと人のキューを調整する

~~~mermaid
flowchart LR
  A[全チャネルの問い合わせ] --> B[AI支援と評価]
  B --> C[自動解決または技能ルーティング]
  C --> D[人の専門担当者]
~~~

**導入。** Assembledはエージェント応答を継続評価し、必要な技能に応じて自動化と人の間で案件を振り分けます。

**FDEの示唆。** 自動応答率だけを最適化しません。公開事例では50% 超の自動化と90% 超の顧客満足を両立したとされています。
