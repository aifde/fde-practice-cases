# 28. Draftwise：契約作成のための追跡可能な検索

~~~mermaid
flowchart LR
  A[過去の契約と条項] --> B[意味検索と再順位付け]
  B --> C[出典付きの作成提案]
  C --> D[弁護士の確認と交渉]
~~~

**導入。** Draftwiseは生成、埋め込み、再順位付けを組み合わせ、提案した条項の出典まで弁護士が確認できるRAGワークフローを作りました。

**FDEの示唆。** 専門業務では根拠への道筋を標準の体験にします。企業は内部ベンチマークで検索品質が30% 改善したと公表しています。
