# FDE Practice Case Library

[中文](README.md) | English | [日本語](README.ja.md)

A public, source-linked library for AI Forward Deployed Engineers, applied AI engineers, and delivery teams. It focuses on AI that has entered real organizations, data environments, and workflows, rather than model feature lists.

## Read the cases

- [30 categorized cases with workflow diagrams](cases.en.md)
- [中文案例库](cases.md)
- [日本語の事例集](cases.ja.md)

Every case includes a Mermaid diagram, the operating problem, the delivery shape, publicly reported evidence, and a reusable FDE lesson.

## Scope

- FDE-style describes a delivery method: work with domain experts, connect production systems, test in the workflow, measure outcomes, and iterate. It does not claim that every organization uses an official FDE job title or team model.
- Sources are public materials from organizations including NHS England, Palantir, OpenAI, Anthropic, and Cohere. Published metrics are not independent audits.
- Mermaid diagrams are used instead of republishing copyrighted marketing images.
- This is an independent community resource. It is not affiliated with or endorsed by any AI laboratory, employer, customer organization, or organization cited in the cases.

## Contribute

Please add only cases with a publicly accessible primary source. A useful contribution identifies the business owner, production workflow, connected data or systems, human-control point, operating metric, and original source.
