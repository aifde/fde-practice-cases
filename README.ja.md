## 本資料における高い確信度

公開情報が次の条件のすべて、または大半を満たす場合だけ掲載します。

1. 顧客または運用組織と、AI導入パートナーの名前が明らかである。
2. 一般的な協業発表ではなく、具体的なAI業務フローがある。
3. 本番利用、実運用への展開、または正式公開済みの企業製品である。
4. 統合、評価、ガバナンス、利用定着、測定結果の証拠がある。
5. 可能な限り、顧客、提供者、規制提出資料、製品資料、独立報道を組み合わせる。
6. AI FDEが何を発見し、構築し、評価し、統制し、改善するかを説明できる情報量がある。

高い確信度は、すべての数値が独立監査済みという意味ではありません。各事例では、確認できる導入事実、参加企業が発表した成果、まだ証明されていない主張を分けています。

## 範囲と独立性

- FDE型は導入方法を指します。利用者の業務に入り、本番システムへ接続し、実際の流れで構築と評価を行い、リスクを管理し、成果を測って改善します。すべての組織が正式なFDE職種を使ったという意味ではありません。
- 医療、政府のサイバーセキュリティ、銀行、ライフサイエンス、顧客サポート、クラウド基盤、法律技術、企業アプリケーションを扱います。
- 著作権のあるマーケティング画像を転載せず、Mermaid図でシステムと人の流れを説明します。
- 本資料は独立したコミュニティ資料です。OpenAI、Amazon、Anthropic、Cohere、Palantir、Decagon、掲載企業による推薦や提携を示しません。

## 貢献

公開情報に裏付けられた本番事例だけを提案してください。事業責任者、導入パートナー、実業務、接続システム、人による統制点、評価方法、運用成果、証拠の限界を示す必要があります。
