各事例にはMermaidのフロー図、運用課題、導入の形、公開された根拠、再利用可能なFDEの学びを含めています。

## 範囲

- FDE型は提供方法を指します。業務専門家と課題を定義し、本番システムへ接続し、業務内で試し、成果を測り、改善を続けます。すべての組織が公式なFDE職種やチームを持つという意味ではありません。
- 情報源はNHS England、Palantir、OpenAI、Anthropic、Cohereなどの公開資料です。公表された数値は独立監査ではありません。
- 著作権のあるマーケティング画像を転載せず、Mermaidの図でシステムと人の関係を表します。
- 本資料は独立したコミュニティ資料であり、AI研究所、雇用主、顧客組織、事例中の組織との提携や推薦を示すものではありません。

## 貢献

公開された一次情報がある事例を追加してください。事業責任者、本番の業務フロー、接続したデータやシステム、人による統制点、運用指標、原資料を示すと有用です。
